# Quelques notions préalables

Les données brutes de ce projet sont extraites d'un travail de recherche mené dans le cadre du mémoire de master 1 MLCC (Mondes littéraires, création critique) à Paris 8. Il porte sur les représentations artistiques dans les fanfictions dérivées de la série Hannibal (2013-2015, NBC, créée par Brian Fuller). Les fanfictions sont des écrits dits transformatifs ou dérivés.<br/>Une fanfiction est un texte d’une longueur variable (de quelques dizaines de mots, jusqu’à plusieurs centaines de milliers), écrit par un-e auteurice, disponible sur des sites spécialisés. Ils ont la spécificité d’être transformatifs ou dérivés : ils partent généralement d’un média source (livre, film, série TV…) et transforment cette source, ou présentent une proposition dérivée de celle-ci. Dans le cas présent, cette production médiatique est la série Hannibal, elle-même dérivée d'ouvrages de Thomas Harris comme Red Dragon et The Silence of the Lambs.<br/>Le corpus fondant notre recherche est composée de plus de deux-cent fanfictions, desquelles on observe le texte lui-même et les paradonnées (data dépendant du site d'archives), ayant toutes pour point commun de traiter d'art ou de représentation artistique. L'enjeu est d'étudier les modalités et les formes de ces manifestations, en les replaçant dans leur contexte matériel de production, ainsi que de noter la singularité de ce phénomène de pictophagie qui s'articule avec l'obscédant topos de l'anthropophagie issu du matériau source.<br/>Le corpus est tiré du site ArchiveOfOurOwn, projet fondé par l'organisation à but non-lucratif Organisation for Transformative Works (OTW). Dans le cas de notre recherche, les données des fanfictions dépendant du site sont les suivantes : 
- titre de la fanfiction
- pseudonyme de l'auteur
- date de dernière publication
- langue d'écriture
- nombre total de mots
- *hits*, soit nombre de consultation de la page
- *kudos*, soit nombre de mention "j'aime"
- *category* ou catégorie, type de relations
- *warning* ou avertissement de contenu sensible
- *rating* ou classification
- fandom, soit production médiatique source
- statut de complétion
- *tags*, ou mots-clefs, indiquant les personnages, les relations des personnages et les tropes ou thèmes que l'auteurice souhaite faire figurer

Enfin, nous soulignons que nous utiliserons "fanfiction" pour désigner l'assemblage du texte lui-même et des paradonnées, et que ces deux aspects seront parfois distingués, notamment dans les méthodologies de recherche mobilisées.
